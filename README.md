# peralihan-energi-menjadi-energi-listrik-karena-kenaikan-bbm
Pada project ini, saya melakukan web scraping untuk mengambil data berita dari website. Topik yang diambil berkaitan dengan energi dan transportasi, termasuk isu kenaikan harga BBM dan peralihan ke energi alternatif seperti kendaraan listrik.
