# Web Scraping Berita BBM
## Deskripsi
Project ini merupakan implementasi web scraping menggunakan Python untuk mengambil data berita dari website. Data yang diambil berkaitan dengan isu energi, termasuk kenaikan harga BBM dan peralihan ke energi alternatif.

Proses scraping dilakukan secara otomatis dengan mengambil informasi berupa judul berita, isi berita, dan link artikel dari halaman web.

## Tujuan
- Mengambil data dari website secara otomatis
- Menyimpan data dalam format JSON
- Memahami dasar penggunaan web scraping menggunakan Python

## Tools & Library
- Python
- requests
- BeautifulSoup
